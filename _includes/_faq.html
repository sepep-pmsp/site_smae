Como posso criar um perfil de usuário e usar o SMAE para gerenciar meus projetos?

Para obter acesso ao SMAE é preciso fazer uma solicitação através do e-mail smae.prefeitura.sp@fgv.br informando nome completo, e-mail, órgão de atuação e perfis de acesso. Você receberá um mensagem no email informado com o link para a tela de login do sistema e uma senha temporária para seu primeiro acesso.

Não consigo visualizar todos os módulos de SMAE.

Quando um novo usuário é cadastrado no sistema, são atribuídos à ele alguns perfis de acesso. Você pode visualizar, filtrar e até mesmo modificar dados no SMAE dependendo do seu perfil de acesso. Caso você queira ter acesso a outras módulos do SMAE, faça uma solicitação através do e-mail smae.prefeitura.sp@fgv.br.

Sou administrador do sistema e preciso cadastrar um novo usuário.

Uma das atribuições do perfil de administrador do sistema é possibilidade de criar, editar, bloquear e excluir usuários. Para criar um novo usuário, clique no módulo Usuários no menu do sistema > Na parte superior da página, clique em Novo Usuário > Preencha os campos E-mail, Nome Completo, Lotação e Órgão (os campos com * são obrigatórios) > defina os perfis de acesso do novo usuário (por padrão adotamos os perfis de Ponto Focal e Orçamento-Metas) > na parte inferior da página clique em Criar Cadastro para salvar. Clique aqui para assistir um tutorial sobre a criação de novos usuários no SMAE.

E se eu esquecer a minha senha?

É muito comum que se esqueça das senhas, por isso o SMAE tem uma opção de "Esqueci minha senha", onde enviará uma senha temporária para o e-mail cadastrado. Clique no "Esqueci minha senha"; informe o e-mail cadastrado e utilize a senha enviada por e-mal. Depois registre sua nova senha para garantir seu acesso!
